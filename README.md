Pour le carnet de suivi sportif je vais créer une structure d'une séance comme demandé avec 
le nom de l'activité ;
la durée ;
l'intensité ;
le nombre de calories ;
le type d'activité.

Je pense ensuite faire un tableau pour stocker les séances et ensuite le programme affichera le menu comme demandé avec
ajouter une séance ;
supprimer une séance avec son nom ;
afficher toutes les séances ;
 mettre à jour toutes les séances selon leur type ;
 trier les séances par durée ;
 trier les séances par calories ;
 quitter le programme.

 Ensuite pour la mise à jour d'une séance il faut que je fasse 
 une séance de course ajoutera 50 calories ;

 une séance de musculation augmentera l'intensité de 1 ;

 une séance de vélo ajoutera 30 calories et augmentera l'intensité de 1.

 Je vais utiliser une structure Seance pour représenter une séance.
 
 Mon programme va etre separe en plusiers fonctions 

 Une fonction pour ajouter une séance.
 Une fonction pour supprimer une séance.
 Une fonction pour afficher les séances.
 Une fonction pour mettre à jour les séances.
 Une fonction pour trier les séances par durée.
 Une fonction pour trier les séances par calories.
 Une fonction pour afficher le menu principal.

Dans mon programme je vais utiliser une boucle pour que l'utilisateur choisisse jusqu'a qu'il veuille quitter.
